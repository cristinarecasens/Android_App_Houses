<h1>Real Estate App (In Development)</h1>

<p>This is a <strong>work-in-progress application</strong> for a hypothetical real estate agency. The app is designed to showcase properties and provide an interactive user experience.</p>

<h2>Features</h2>
<ul>
        <li>Image Slider – Browse property images easily.</li>
        <li>Property Submission Form – Add new properties through a simple form.</li>
        <li>Updatable Property List – View and update the list of available properties.</li>
        <li>Location Map – Visualize property locations on an interactive map.</li>
</ul>
